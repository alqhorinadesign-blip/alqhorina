Setup Firebase

Hubungkan ExamBrowser ke Firebase agar data tersimpan online & link bisa disebar ke siswa

1
Buat project Firebase (gratis) Buka console.firebase.google.com → klik "Add project" → beri nama → selesaikan wizard.
2
Aktifkan Realtime Database Di sidebar kiri → Build → Realtime Database → "Create database" → pilih lokasi terdekat → mulai dalam test mode (30 hari) lalu ganti rules ke production.
3
Dapatkan konfigurasi Di Project Settings (ikon ⚙️) → General → scroll ke bawah → Your apps → klik </> (Web) → daftarkan app → salin firebaseConfig.
4
Host file ini (agar link bisa disebar) Upload file exambrowser.html ke Netlify Drop (gratis, drag & drop) → dapat link seperti https://xyz.netlify.app → link inilah yang dibagikan ke siswa.
Konfigurasi disimpan di browser ini. Setelah terhubung, semua data tersimpan di Firebase.
